\# Kenya Services Access Centre



A comprehensive marketplace platform connecting employers, job seekers, and service providers in Kenya with M-PESA integration.



\## Features



\- \*\*Employers\*\*: Post jobs (KES 400 entry + KES 300/month subscription)

\- \*\*Job Seekers\*\*: Find jobs (KES 50/view requirements, KES 100/employer details, KES 50/CV upload)

\- \*\*Service Providers\*\*: Offer services (KES 100 to connect)

\- \*\*M-PESA Integration\*\*: Via TUMA API

\- \*\*Admin Dashboard\*\*: Full control over users, jobs, transactions, and settings

\- \*\*Real-time Notifications\*\*: WebSocket for instant updates



\## Tech Stack



\- \*\*Backend\*\*: Node.js + Express.js

\- \*\*Database\*\*: PostgreSQL

\- \*\*Frontend\*\*: HTML5 + TailwindCSS + JavaScript

\- \*\*Payments\*\*: TUMA API (M-PESA)

\- \*\*Real-time\*\*: Socket.io
